# README #

### What is this repository for? ###

* A starter kit for js development.
* It includes follows
- webpack
- minify
- uglify
- livereload

### How do I get set up? ###

```
git clone https://hajimenagata@bitbucket.org/hajimenagata/d3es6_starter.git
mv d3es6_starter _repo
cd _repo
npm install
npm start
```
